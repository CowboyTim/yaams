Yet Another Auto Mounter System

Yet Another Auto Mounter System (YAAMS) is an automounter, written in python. It is loosly based on the pymount.py script from Mashi. The original script can only mount devices/volumes/storage devices when they are in /etc/fstab. The original script also doesn't support cdroms being ejected.

This script uses the label, and if not found, the UUID to be mounted under /media. Maybe in the future, both can be merged if needed. It is not my intention to manage yet another auto mounter just for fun. I also like the idea of it being written in a higher level VM language, especially for things like this.

Dependencies

python python-dbus python-gobject

Install

on ubuntu 9.04: 

  $ sudo apt-get install python-dbus python-gobject
  $ sudo python yaams.py -d
